# DevColor Studio

A modern web application for creating, converting, and managing colors and gradients.

## Features
- Color Picker
- Gradient Generator
- HEX/RGB/HSL Converter
- Random Palette Generator
- Copy Colors
- Save Favorites
- Responsive UI

## Technologies
- HTML5
- CSS3
- JavaScript
